How Risk Classification and Glucose Trend Work

Risk Was Classified
Profile + Lifestyle Feature Classification
  • The system combines profile fields and recent logs into one feature vector.
  • Features are normalized using the saved classification scaler.
  • The classifier predicts risk probability and maps it to Low or High risk.
Glucose Trend Works
Next-Value Trend Regression
  • The trend model reads transformed behavioral and biomarker inputs.
  • The regression scaler aligns runtime input shape with trained features.
  • The regressor estimates the next glucose value used for trend insights.
